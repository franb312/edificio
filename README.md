# Edificio Real Estate

Modern real estate website scaffold for showcasing apartments with a strong design focus.

## Available Scripts

- `npm install` — install dependencies
- `npm run dev` — start the local development server
- `npm run build` — build the production site
- `npm run preview` — preview the production build

## Project Structure

- `index.html` — application entry point
- `src/main.jsx` — app bootstrap
- `src/App.jsx` — main landing page and property listings
- `src/index.css` — Tailwind styles and global CSS
